Interitus -- Programmer

See More

Interitus was a collaboration between a friend and myself. He did the art, and I did the programming. It was made with an engine called GameMaker Studio, which uses a language very similar to JavaScript.

The premise of the game is to survive for a set number of days against a horde of zombies that gets larger as the days go on. In order to surivve, the player must carefully manage his or her resources, including food, health, ammunition, and scrap metal.

The Infi-gen -- Programmer

See More

The Infi-Gen is a proof-of-concept game that I programmed for my Digital Media course final. Like Interitus, it was also programmed using the GameMaker Studio engine.

The main feature of the game is that all of its levels are procedurally-generated by the computer using the A* algorithm. Hence, the game's name: infi(nite)-gen(eration). Notwithstanding being a proof-of-concept, The Infi-Gen is a playable dungeon-based crawler.

WHS Robotics -- Programmer & Graphic Designer

In addition to programming video games, I also am a programmer for my school's robotics team, FTC Team 542. Since we participate in FIRST tournaments, the other programmers and myself do all of our work in Java.

Besides programming, I also serve as the team's treasurer and graphic designer. As the former, I work to procure funds for the team and log its expenses. As the latter, I design club shirts and even created an animatic for the team's 2017 Promote video.

jj11d7t.github.io -- Programmer

See More

This website is also a project of mine, one that was actually started pretty recently. Although HTML and CSS were among the first languages I learned, I did not really think about making my own website until the summer of 2017. It had been a while since I used the two languages, and I had never used Boostrap or JavaScript in tandem with them, but I was excited at the challenge and constructed a workable version within a few days.

As of now, the website is still under construction as I try to figure out the right aesthetic, make the code more efficient, and optimize it for all screen sizes so mobile users can view it too. Nonetheless, the source code for this website is publicly available on Github; feel free to take a look under the hood!
