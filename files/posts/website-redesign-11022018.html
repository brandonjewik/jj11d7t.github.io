Website Redesign

11 February 2018

So the website is new.

First things first, I redid the aesthetic to make it more minimalistic and easy on the eyes. In addition, everything now scales (or should) correctly! Using inspect element to check it from different screen resolutions, it seems like everything should work okay. We'll have to wait and see when I go check on the 4k monitors at school on Tuesday.

I also added some cool particle effects on the home page, courtesy of Vincent Garreau. It looks so nice and polished now. However, I didn't actually change anything on the 'Projects' page. I got rid of the buttons that used to span the top of the page because I realized that it would be impractical to have them as I add more and more projects to the page. Otherwise, the code is almost exactly the same as the old 'Projects' page. Still works and looks good, so why change anything?

No mobile page this time around though. I figure it's too much effort to make one, plus this website should really be viewed on desktop to get the full experience anyway.
